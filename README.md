# LicensePlateRecognition

 Hello everyone, this project includes license plate recognition with OpenCv and saves detected images. I didn't compile because the exe file maybe not work your computer, you must be compile this solution for use.
 
 You need to add OpenCv's include path and library path for compile this solution.
 
 If you want improve this project you can do threshold and bitwise transformation then read the characters of plate with an optic character recongnizer(Tesseract etc.) and save plate image with own plate characters. Good luck everyone :)

 Turkish description:
 Herkese merhabalar, bu proje OpenCv kullanarak kamera görüntüsünden araç plakası tespiti ve kaydedilmesi kodlarını içeriyor. Projeyi derlemeden yüklüyorum çünkü derlenmiş projenin exe dosyası her bilgisayarda çalışmayabilir projeyi derlemeniz ve öyle kullanmanız gerekiyor. 
 
 Projeyi derleyebilmek için önce OpenCv'nin include path ve library path dizinlerini eklemeniz gerekiyor ardından x64 Debug modunda projeyi derleyebilirsiniz.
 
 Projenin geliştirilmesi kapsamında tespit edilen plaka kaydedilmeden önce görüntüye threshold ve bitwise dönüşümü uygulanıp bir optik karakter okuyucu(Tesseract vb.) yardımıyla plakanın karakterleri okunup plakanının kendi ismiyle kaydedilmesi sağlanabilir. Herkese iyi şanslar :)
